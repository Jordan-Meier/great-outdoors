INSTALL
-------
This module is tracking the latest version of the jQuery.LocalScroll
and jQuery.ScrollTo plugin.

The goal is to reach the following files in the libraries directory:

jquery.localScroll/jquery.localScroll.min.js
jquery.localScroll/jquery.localScroll.js
jquery.scrollTo/jquery.scrollTo.min.js
jquery.scrollTo/jquery.scrollTo.js


Go to the libraries directory, eg sites/all/libraries

1. wget https://github.com/flesler/jquery.scrollTo/zipball/master
2. unzip master
3. mv flesler-jquery.scrollTo-* jquery.scrollTo
4. rm master
5. wget https://github.com/flesler/jquery.localScroll/zipball/master
6. unzip master
7. mv flesler-jquery.localScroll-* jquery.localScroll
8. rm master


Enable the module as usual.
